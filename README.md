# Canvas API 스터디

이미지 편집기 구현을 위한 Canvas API 스터디

## 필요 기능

- 캔버스에 이미지 노출/추출
- 확대/축소, 회전, 좌우반전
- 크기 변경
- 보정: 밝기, 대비, 채도, 색온도
- 필터
- 모자이크
- 캔버스를 대상으로 하는 인터렉션: 영역 선택, 이미지 이동 등
- 선택 영역 자르기(crop)
